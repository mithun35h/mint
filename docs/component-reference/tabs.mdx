---
title: Tabs
description: Toggle content using the Tabs component
---

## Example

<Tabs>
  <Tab title="First Tab">
    ☝️ Welcome to the content that you can only see inside the first Tab.
  </Tab>

  <Tab title="Second Tab">
    ✌️ And... here's content that's only inside the second Tab.
  </Tab>

  <Tab title="Third Tab">
    Lastly, here's the third tab at the end.
  </Tab>
</Tabs>

## Usage

You can add any number of tabs.

```jsx
<Tabs>
  <Tab title="First Tab">
    ☝️ Welcome to the content that you can only see inside the first Tab.
  </Tab>

  <Tab title="Second Tab">
    ✌️ And... here's content that's only inside the second Tab.
  </Tab>

  <Tab title="Third Tab">
    Lastly, here's the third tab at the end.
  </Tab>
</Tabs>
```

### Props

<ResponseField name="title" type="string">
  The name of the tab displayed in the title
</ResponseField>

<ResponseField name="children" type="markdown">
  Content inside the tab
</ResponseField>
