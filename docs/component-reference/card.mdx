### Icon Settings

Adjust the icon type using the `iconType` property color by using the `color` property.

<Card title="Colored Card" icon="lightbulb" iconType="duotone" color="#ca8b04">
  Here is a card with a duotone yellow icon
</Card>

```jsx
<Card title="Colored Card" icon="lightbulb" iconType="duotone" color="#ca8b04">
  Here is a card with a duotone yellow icon
</Card>
```

### Adding a URL

Make a card a navigation item by adding the `href` property like you would on an anchor element.

<Card title="Click on me" icon="link" href="/quickstart">
  Clicking on this card brings you to the quickstart page
</Card>

```jsx
<Card title="Click on me" icon="link" href="/quickstart">
  Clicking on this card brings you to the quickstart page
</Card>
```

### Props

<ResponseField name="title" required type="string">
  The title of the card
</ResponseField>

<ResponseField name="icon" type="Icon">
  A [Font Awesome icon](https://fontawesome.com/icons) or SVG code in `icon={}`
</ResponseField>

<ResponseField name="iconType" type="string">
  One of `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands`
</ResponseField>

<ResponseField name="color" type="string">
  The color of the icon as a hex code
</ResponseField>

<ResponseField name="href" type="string">
  The url that clicking on the card would navigate the user to
</ResponseField>

<ResponseField name="children" type="MDX or string">
  Supplemental information for the card under the title (e.g. description,
  examples)
</ResponseField>
